PixelRift is a modern online gaming website where users can browse games, view game details, play one simple browser game, join tournaments, check leaderboards, manage their profile, and contact support. The website will have a dark neon gaming style, simple navigation, responsive pages, and a clean layout that works on desktop, tablet, and mobile.
The project will include all main areas: Home Page, Login and Register, Games Listing, Game Details, Game Play, Tournament, Leaderboard, Profile and Contact, and Admin Dashboard. It will use local sample data with 12 games, 5 tournaments, 15 leaderboard players, and 3 sample users, and it will run locally without external service dependency.
