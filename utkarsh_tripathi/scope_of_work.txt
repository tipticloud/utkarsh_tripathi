The scope of work is to build a complete frontend online gaming website named PixelRift. The website will let users browse games, view game details, play one simple browser game, join tournaments, check leaderboards, manage a profile, and contact support.
The project will include 9 main areas: Home Page, Login and Register, Games Listing, Game Details, Game Play, Tournament, Leaderboard, Profile and Contact, and Admin Dashboard. It will use local sample data with 12 games, 5 tournaments, 15 leaderboard players, and 3 sample users.
The work also includes responsive design, safe gaming themed assets, clear navigation, form validation, login state, tournament join state, profile update, contact form success message, and admin sample actions to add, edit, and delete games and tournaments. The website must run locally and must not use copied game assets or restricted content.
